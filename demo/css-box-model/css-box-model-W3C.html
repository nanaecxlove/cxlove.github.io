W3C盒模型总结：

  • W3C盒模型Width = contentWidth + pdding + margin + border；

border区背景色是否填充

  • 非IE浏览器下，div的背景颜色充满了padding和border。border透明时，可发现border区域填充了背景色。
  • IE8和IE9在标准模式下，和非IE浏览器渲染一致，border区域填充背景色；
  • IE6 IE7标准模式下，border区域不填充背景色；
